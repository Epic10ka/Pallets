        Sobre:
Fiz esse projeto para facilitar o trabalho da minha contagem de pallets no meut rabalho.


        Conceitos aplicados:
-Loops

-Funções

-indentação e aninhamento

-Tratamento de erros

-Acumuladores

-Dicionário

-Retorno de valores

          Funcionalidades: 

-O usuário pode escolher o tipo de pallets

-O usuário escolhe se contará em fileiras ou em blocos separados

-A opção 'Ver resumo' mostra ao usuário as informações das pallets

-O usuário pode resetar as contagens

           Como executar o projeto:

-Clone ou baixe esse repositório: [https://github.com/Epic10ka/Gerenciador-de-Estacionamento](https://github.com/Epic10ka/Pallets)

-Abra a pasta do projeto

-Execute o arquivo `main.py` e utilize-o
